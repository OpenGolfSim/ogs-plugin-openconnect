A plugin to allow other apps and devices to send shot data to OpenGolfSim over the OpenConnect v1 API.


## Installation

To install this plugin, download this repo as a ZIP file (click the green **Code** button above and select **Download Zip**). Then extract the folder to your plugins folder location for your platform.


#### Windows

On Windows, the plugins folder should be found at:

```
%USERPROFILE%\AppData\Roaming\opengolfsim-desktop\plugins\
```

> Tip: You should be able to copy + paste the above into your file explorer address bar.

#### MacOS

On MacOS, the plugins folder should be found at:

```
~/Library/Application Support/opengolfsim-desktop/plugins/
```

> Tip: In the Finder under the **Go** menu, select **Go to Folder** and copy + paste the above to the text box.
